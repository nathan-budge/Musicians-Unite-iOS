Tutorial

The LE MeterWorx™ PitchDetector SDK is truly straightforward to use. It is wrapped up in a single class: LE::Meter::PitchDetector:

// Create and setup the PitchDetector:
LE::Meter::PitchDetector pitchDetector;
if ( !pitchDetector.setup( sampleRate, numberOfChannels ) )
{
// <out of memory>
}
// Process the input data:
while ( <data> )
{
pitchDetector.process
(
pInputData,
samplesToProcess
);
// Examine intermediate results:
// Overall pitch:
currentPitch = pitchDetector.pitch();
// Per-channel pitch:
currentPitchChannel_0 = pitchDetector.pitch( 0 );
currentPitchChannel_1 = pitchDetector.pitch( 1 );
// etc.
}
